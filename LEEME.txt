MAPA INTERACTIVO DE GESTIÓN TERRITORIAL
Versión 3.5 Web (Navegador)

═══════════════════════════════════════════════════════════

DESCRIPCIÓN
Aplicación web para la gestión visual de asignaciones provinciales.
Funciona directamente en el navegador sin necesidad de instalación.

CARACTERÍSTICAS
✓ Gestión Visual: Asignación por colores y responsable
✓ Panel de Datos: Población actualizada (INE) y desglose
✓ Exportación: Excel detallado e imagen PNG
✓ Persistencia: Los datos se guardan automáticamente en el navegador

═══════════════════════════════════════════════════════════

INSTRUCCIONES DE USO

OPCIÓN 1: Doble Click (Recomendado - CIERRE AUTOMÁTICO)
---------------------------------------------------------
1. Doble click en "ABRIR_MAPA.vbs"
2. Espera 2-3 segundos
3. Se abrirá automáticamente en tu navegador
4. NO aparece ventana negra (servidor oculto)

✓ CIERRE AUTOMÁTICO: Cuando cierres el navegador, el servidor se 
  detiene automáticamente. ¡No necesitas hacer nada más!

OPCIÓN 2: Con Python (Aparece consola - Cierre manual con Ctrl+C)
-----------------------------------------------------
1. Doble click en "INICIAR.bat"
2. Se abrirá automáticamente en tu navegador
3. NO cierres la ventana negra (es el servidor)
4. Cuando termines, cierra el navegador y presiona Ctrl+C en la ventana

OPCIÓN 2: Sin Python (Alternativa)
-----------------------------------
1. Abre Chrome o Edge
2. Arrastra el archivo "index.html" al navegador
3. Acepta los permisos si te los pide

═══════════════════════════════════════════════════════════

DATOS Y PERSISTENCIA

• Los datos se guardan automáticamente en el navegador (localStorage)
• Son específicos del navegador y perfil que uses
• Para transferir datos entre PCs: usa la exportación a Excel
• Para hacer backup: Exporta a Excel regularmente

═══════════════════════════════════════════════════════════

REQUISITOS

• Navegador moderno (Chrome, Edge, Firefox)
• Python 3.x (solo para OPCIÓN 1)
  - Verifica si lo tienes: abre CMD y escribe "python --version"
  - Si no lo tienes, usa OPCIÓN 2
